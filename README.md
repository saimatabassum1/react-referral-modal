# React Referral Modal

A pixel-perfect, fully responsive referral modal built with React from a Figma design, carefully replicating the original UI across mobile and desktop screens.
---
# Live Demo/ Preview
[View Live Project](https://react-referral-modal.vercel.app/)

# Features

- Responsive design (mobile + desktop)
- Clean and modern UI
- Figma to React implementation
- Smooth layout and spacing
- Reusable modal component
-  Deployed with Vercel
---

# Tech Stack

- React
- CSS
- Vite
- Vercel
